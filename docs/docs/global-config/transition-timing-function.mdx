## Global Transition Timing Function


:::info

The animation of this library is done using CSS transition.

This page describes how to configure the
default transition timing function. This affects the timeline progress bar's transition and when a prop's transition is not provided in its frame data.

(see [Prop Transition Timing Function](frame-animation-speed) for prop's frame based transition timing function)

:::


### Default Behavior

By default, this value is set to `linear`.

Check how the timeline's progress bar and props animate:

<SceneOnlyShort {...sceneDefaultTransitionFunction} />

### Setting Transition Timing Function

You can set **transitionTimingFunction** to any css transition-timing-function:

<GeneratorConfigBlock
    generator={
        `new GlobalConfigGenerator().transitionTimingFunction('ease-in-out')`}
    config={
        `config.transitionTimingFunction = 'ease-in-out'`} />

<SceneOnlyShort {...sceneTransitionFunctionEaseInOut} />
