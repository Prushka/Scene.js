### Library comes with the following themes:

* light (default theme)
* dark
* dark-classic

:::tip

All scenes in this theme have scope: **container**.

That's why changing the theme of any scene instance in this page only affects that scene.

You can change this behavior to make it global (see [Theme Scope](theme-scope))

:::

### Select a theme as default theme

<GeneratorConfigBlock
    generator={"new GlobalConfigGenerator().defaultTheme('dark')"}
    config={
`config.defaultTheme = 'dark'
`} />


<SceneOnlyShort {...sceneDefaultDarkTheme}/>
