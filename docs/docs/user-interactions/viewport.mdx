## Viewport

* Users can use mouse or touch to **drag the viewport**
* Users can use mouse wheels to **zoom in or zoom out**
* Users can click any prop to open its **Prop Dialog**

:::caution

To prevent users from **zooming in or out too far**, developers can configure the max/min zoom distance in the global config.

Developers can also configure the **zoom step** in global config.

:::
